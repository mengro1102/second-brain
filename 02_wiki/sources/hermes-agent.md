# Hermes Agent: 오픈소스 AI 에이전트

## Kernel

NousResearch가 만든 오픈소스 AI 에이전트. MIT 라이선스. 메모리 기반 동작으로 세션 간 맥락 유지. 텔레그램 연동 가능.

## 핵심 특징

1. **MIT 라이선스**: 완전 오픈소스, 상업적 사용 가능
2. **메모리 기반 동작**: 세션 간 맥락을 유지하여 연속적 대화 가능
3. **텔레그램 연동**: 메신저를 통한 에이전트 접근
4. **NousResearch 개발**: 오픈소스 AI 커뮤니티의 핵심 조직

## 캡처 맥락

- 목적: AI 에이전트 생태계 탐색
- 연결: 세컨드 브레인과 에이전트 연동 가능성 검토
- 다음 단계: 로컬 설치 테스트, MCP 연동 가능성 확인
