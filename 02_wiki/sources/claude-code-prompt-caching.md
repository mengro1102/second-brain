# Claude Code 구축에서 얻은 교훈: 프롬프트 캐싱

## Kernel

장기 실행 에이전트에서 프롬프트 캐싱은 이전 라운드트립의 연산을 재사용해 지연/비용을 줄이는 핵심 기술. Claude Code 전체 아키텍처가 이를 중심으로 설계됨.

## 핵심 내용

1. **프롬프트 캐싱**: 이전 라운드트립의 KV 캐시를 재사용하여 중복 연산 제거
2. **아키텍처 설계 원칙**: Claude Code는 캐싱 효율을 극대화하도록 전체 구조 설계
3. **비용/지연 절감**: 장기 실행 에이전트에서 누적 비용과 응답 지연 대폭 감소
4. **에이전트 최적화**: 멀티턴 에이전트 시스템의 핵심 성능 병목 해결

## 캡처 맥락

- 목적: 에이전트 아키텍처 최적화 기법 학습
- 연결: 에이전트 시스템 설계 시 성능 최적화 참고
- 다음 단계: 자체 에이전트 구축 시 캐싱 전략 적용 검토
