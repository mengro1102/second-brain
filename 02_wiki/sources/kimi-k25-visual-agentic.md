# Kimi K2.5: Visual Agentic Intelligence

## Kernel

Kimi K2.5는 시각적 에이전틱 지능을 갖춘 멀티모달 모델. 비전 기반 에이전트 태스크에서 강력한 성능.

## 핵심 내용

1. **Visual Agentic Intelligence**: 시각 정보를 기반으로 에이전트 행동을 수행
2. **멀티모달**: 텍스트 + 비전 통합 처리
3. **에이전트 태스크**: GUI 조작, 웹 브라우징 등 시각 기반 자율 행동
4. **Kimi 시리즈**: Moonshot AI의 플래그십 모델 라인업

## 캡처 맥락

- 목적: 멀티모달 에이전트 모델 동향 추적
- 연결: 비전 기반 에이전트의 실용성 평가
- 다음 단계: 논문 딥리딩, 에이전트 벤치마크 비교
