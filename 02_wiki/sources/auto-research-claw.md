# AutoResearchClaw: 완전 자율 AI 연구 에이전트

## Kernel

아이디어를 기반으로 학술 논문을 완성하는 완전 자율 AI 연구 에이전트. 코드 구현부터 논문 작성까지 자동화.

## 핵심 특징

1. **완전 자율**: 아이디어 입력만으로 논문 완성까지 자동 수행
2. **코드 구현 자동화**: 실험 코드 생성 및 실행
3. **논문 작성 자동화**: 구조화된 학술 논문 형식으로 결과 정리
4. **End-to-End 파이프라인**: 아이디어 → 문헌조사 → 실험 → 작성

## 캡처 맥락

- 목적: 연구 자동화 도구 탐색
- 연결: AI 연구 생산성 극대화 가능성 검토
- 다음 단계: 실제 사용 가능성 평가, 한계점 파악
