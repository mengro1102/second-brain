# MiroFish: 집단 지능 시뮬레이션 엔진

## Kernel

문서 하나를 넣으면 수천 명의 가상 인간을 생성하여 토론/논쟁/예측을 시뮬레이션하는 집단 지능 엔진. CAMEL-AI + OASIS 기반. GitHub 글로벌 트렌딩 1위.

## 핵심 특징

1. **단일 문서 입력**: 문서 하나로 수천 명의 가상 인간 생성
2. **집단 지능 시뮬레이션**: 토론, 논쟁, 예측 등 다양한 상호작용
3. **CAMEL-AI + OASIS 기반**: 멀티 에이전트 프레임워크 활용
4. **GitHub 글로벌 트렌딩 1위**: 높은 커뮤니티 관심

## 캡처 맥락

- 목적: 집단 지능 시뮬레이션 기술 탐색
- 연결: 연구 아이디어 검증에 활용 가능성
- 다음 단계: 실제 사용 사례 분석, 연구 적용 시나리오 설계
