# UNIST: AI 학습 데이터에 설명문 붙여 딥러닝 블랙박스 해독

## Kernel

UNIST 연구진이 학습 데이터를 인간이 이해할 수 있는 설명문으로 변환해 AI 블랙박스를 해독하는 방법 제시.

## 핵심 내용

1. **데이터 기반 설명**: 모델 내부가 아닌 학습 데이터에 설명문을 부착하는 접근
2. **블랙박스 해독**: 딥러닝 모델의 의사결정 과정을 인간이 이해 가능한 형태로 변환
3. **UNIST 연구**: 국내 연구진의 XAI 기여

## 캡처 맥락

- 목적: XAI 연구 동향 추적
- 연결: AI 모델의 신뢰성·해석 가능성 확보 방법론
- 다음 단계: 논문 원문 확인, Surrogate 모델 해석에 적용 가능성 검토
