## 핵심 특징

1. **경량**: Claude Code 대비 44배 가벼운 코드베이스
2. **Python 기반**: 접근성 높은 언어로 구현
3. **오픈소스**: 커스터마이징 및 확장 용이
4. **HKUDS 개발**: 홍콩대학교 데이터 사이언스 연구팀

- 연결: 개인 에이전트 구축 시 기반 프레임워크 후보
