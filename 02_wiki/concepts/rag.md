# RAG (Retrieval-Augmented Generation)

LLM을 건드리지 않고, 질문 시 관련 문서를 검색해서 컨텍스트에 주입하는 방식.

## Kernel

파인튜닝 = 모델 행동 변경. RAG = 모델이 읽는 자료 변경. 매일 변하는 개인 지식에 적합.

## LLM-Wiki와의 차이

RAG: 매번 원문에서 재추출. LLM-Wiki: 지식을 한 번 컴파일하고 점진적 축적. 두 접근은 상호 보완적.
