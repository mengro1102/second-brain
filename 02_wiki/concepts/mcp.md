# MCP (Model Context Protocol)

Anthropic이 2024년 공개. LLM이 외부 도구와 데이터 소스를 직접 호출하는 표준 인터페이스.

## Kernel

기존: 플러그인이 LLM 호출 (LLM = 수동). MCP 이후: LLM이 능동적 에이전트로 도구 직접 호출.
