# output/ 폴더 규칙

이 폴더는 위키에서 파생된 최종 결과물을 저장한다.

## 구조

- `blog/` — GitHub Pages 블로그용 포스트
- `portfolio/` — 포트폴리오 페이지 (연구 성과, 프로젝트 기록)
- `slides/` — 발표자료 (Marp 마크다운 등)

## 규칙

1. 결과물은 wiki/ 페이지를 기반으로 생성한다
2. 원본 wiki 페이지를 frontmatter의 `wiki_source`에 명시한다
3. 블로그 포스트는 공부/포트폴리오/개인 카테고리로 분류한다
4. 톤: 간결하고 핵심 중심. 기술 용어는 영어 병기.
